ExLinks

CLICK HERE TO INSTALL EXLINKS

ExLinks is a userscript created for Chrome and Firefox that makes browsing Sad Panda threads a much nicer experience. Instead of having a list of plain links to open one at a time, ExLinks will save you a ton of time by turning the links into something useful. Let this picture act as a demonstration:

Installation (Chrome)

  1. Modify your Chrome shortcut to include --enable-easy-off-store-extension-install and relaunch it.
  2. Install userscript. Alternatively:
  3. Download userscript to your computer.
  4. Drag it to the Extensions page.

Installation (Firefox)

  1. Make sure you have Greasemonkey.
  2. Install the script.

Compatibility

ExLinks is compatible with both vanilla 4chan and 4chan X, as well as any linkification plugins. Stylistically ExLinks is also compatible with OneeChan and 4chan Style Script.

Troubleshooting

If something seems to be failing, make sure all of your software is up to date. ExLinks has been tested with Chrome 21 and Firefox 14, so if you're running anything lower update first and see if that fixes the issue.
