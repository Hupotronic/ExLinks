Exlinks

A userscript to make E-Hentai & ExHentai links on 4chan & Foolz archive more useful.

ExLinks is a userscript created for Chrome and Firefox that makes browsing Sad Panda threads a much nicer experience. Instead of having a list of plain links to open one at a time, ExLinks will save you a ton of time by turning the links into something useful. Let this picture act as a demonstration:

For users of HTTPS Everywhere (or similar)

ExLinks is only guaranteed to work properly over HTTP due to the E-Hentai API being HTTP-only, and as such, if you open any threads where ExLinks would do something, it will automatically redirect you from HTTPS to HTTP to work properly. If you use HTTPS Everywhere, this can lead to infinite HTTPS->HTTP->HTTPS redirection. As such, you will need to disable the plugin (or any similar HTTPS redirection plugins) for 4chan in order to ensure that ExLinks will work properly. See here for instructions.

Installation (Chrome)

  1. Make sure you have Tampermonkey.
  2. Install the userscript by clicking the big install link above.

Installation (Firefox)

  1. Make sure you have Greasemonkey.
  2. Install the userscript by clicking the big install link above.

Troubleshooting

If something seems to be failing, make sure all of your software is up to date. Guaranteed minimums are Chrome 21 and Firefox 14, but you really should be always running the latest stable versions if possible.

Also, make sure you don't have a pre-2.0.0 version (1.7.4 or older) of ExLinks installed, as it will conflict with ExLinks 2.X.X.
